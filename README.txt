依赖库: gym, seabon(均为anaconda包含)

## 声明: 由于本机算力不足和TensorFlow框架能力较差, DQN部分代码是网上的资料,并非自己完成. 只用于简单的方法对比. 

文件结构:
.\task1_3methods.py 任务一的三种方法文件,可以指定文件中的index来选择不同方法.
.\task2_3methods.py 任务二的三种方法文件,可以指定文件中的index来选择不同方法.
.\task1_parameter_compare.py 任务一的方法对比文件,需要更改各个参数的取值来对比三种方法
.\task2_parameter_compare.py 任务二的方法对比文件,需要更改各个参数的取值来对比三种方法
.\DQN.py 非自己完成的DQN实现代码,用于方法对比.
.\task1_method_compare.py 任务一种的三个方法放在一起对比的文件.
